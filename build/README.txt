Nuget Stuff
===========

* Install [nuget.exe][1]

* Run: nuget update

* Run: NuGet SetApiKey <API-KEY>


Creating the package
--------------------

* Change version number in Solutionversion.cs and synoptic.nuspec

* Run nuget_pack.bat

* Login and upload to nuget.org


   [1]: http://docs.nuget.org/docs/creating-packages/creating-and-publishing-a-package
